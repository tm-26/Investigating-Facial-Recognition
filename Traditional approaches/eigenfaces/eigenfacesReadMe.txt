To run eigenfaces.py make sure the following are installed:
python >= 3.8.5
opencv >= 4.0.1 (pip install opencv-python)
numpy >= 1.19.2 (pip install numpy)
pandas >= 1.1.5 (pip install pandas)
scikit-learn >= 0.23.2 (pip install scikit-learn)

eigenfaces.py takes the following parameters:
argv[0] = number of iterations
